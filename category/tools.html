基本设置

  • 显示行号 Editor -> General -> Appearance -> Show line numbers

  • 用*标识编辑过的文件
    Editor -> General -> Editor Tabs -> Mark modified tabs with asterisk

  • 文件说明模板 Editor -> Code Style -> File and Code Templates -> Includes

  • 修改字体 Editor -> Colors & Fonts -> Font

  • 资源文件相关 Editor -> File Encodings
